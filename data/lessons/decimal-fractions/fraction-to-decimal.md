## איך עושים זאת?
1. נבדוק אם אפשר להרחיב את השבר כך שהמכנה יהפוך ל-$10$, $100$ או $1000$.
2. נמצא במה צריך להכפיל את המכנה כדי להגיע לחזקה של $10$, ונכפיל באותו מספר גם את המונה.
3. נכתוב את המספר במונה ונשים את הנקודה לפי המכנה: מכנה $10$ — ספרה אחת אחרי הנקודה; מכנה $100$ — שתי ספרות; מכנה $1000$ — שלוש ספרות.
4. אם המכנה לא מתחלק יפה ב-$10$, $100$ או $1000$, אפשר פשוט לחלק את המונה במכנה.

## דוגמה
נמיר את $\dfrac{7}{40}$ לשבר עשרוני.

- נשאל במה להכפיל את $40$ כדי להגיע ל-$1000$. מחשבים: $1000 \div 40 = 25$.
- מכפילים מונה ומכנה ב-$25$: $\dfrac{7 \times 25}{40 \times 25} = \dfrac{175}{1000}$.
- מכנה $1000$ אומר שיהיו שלוש ספרות אחרי הנקודה: $0.175$.

$$\dfrac{7}{40} = \dfrac{175}{1000} = 0.175$$

## לשים לב
חשוב לזכור שמכפילים גם את המונה וגם את המכנה באותו מספר, אחרת מקבלים שבר שונה. שגיאה נפוצה היא להכפיל רק את המכנה כדי "להגיע ל-$1000$" ולשכוח את המונה. כשמכפילים את שניהם השבר נשאר באותו ערך, רק הצורה שלו משתנה.
