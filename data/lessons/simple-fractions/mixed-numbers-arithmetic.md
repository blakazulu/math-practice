## איך עושים זאת?
1. נמיר כל מספר מעורב לשבר לא־אמיתי. הנוסחה: מכפילים את החלק השלם במכנה, מוסיפים את המונה, וכותבים את הסכום מעל אותו מכנה.
2. אם פועלים חיבור או חיסור — נביא את השברים למכנה משותף, ואז נחבר או נחסר את המונים.
3. אם פועלים כפל — נכפול מונה במונה ומכנה במכנה.
4. אם יש סוגריים או פס שבר ארוך, נסיים קודם את הביטוי שבסוגריים (או בנפרד את המונה ואת המכנה), ורק בסוף נשלב.
5. בסוף נחזיר את התוצאה למספר מעורב אם המונה גדול מהמכנה.

## דוגמה
נחשב את $1\dfrac{1}{4} + \dfrac{1}{4}$.

- נמיר את $1\dfrac{1}{4}$ לשבר לא־אמיתי: $\dfrac{1 \times 4 + 1}{4} = \dfrac{5}{4}$.
- המכנים כבר זהים, אז מחברים את המונים: $\dfrac{5}{4} + \dfrac{1}{4} = \dfrac{6}{4}$.
- מצמצמים ב-$2$: $\dfrac{6}{4} = \dfrac{3}{2} = 1\dfrac{1}{2}$.

$$1\dfrac{1}{4} + \dfrac{1}{4} = 1\dfrac{1}{2}$$

## לשים לב
טעות נפוצה היא לפעול בנפרד על השלמים ועל השברים בלי לבדוק. לדוגמה, בחיסור $3\dfrac{1}{4} - 2\dfrac{3}{4}$, אם נחסר רק שלמים ושברים בנפרד נקבל $1$ מינוס $\dfrac{2}{4}$, וזה לא **שבר תקין**. הדרך הבטוחה היא להפוך לשברים לא־אמיתיים ואז לחסר.
