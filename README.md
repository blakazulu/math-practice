# חשבונייה

תרגול חשבון מתקדם לכיתות ו׳ — הכנה לבחינת הקבלה לתוכניות הצטיינות במתמטיקה.

497 שאלות מקוריות, 23 נושאים, 7 מבחנים שלמים לדוגמה. תרגול חופשי לפי נושא או מבחן מלא בסגנון הבחינה האמיתית. הכל בעברית, חינם, נשמר במכשיר שלכם בלבד.

מקור החומר — [התוכניות לנוער מוכשר במתמטיקה, yuni.co.il](https://yuni.co.il/).

## איך זה עובד

**תרגול לפי נושא.** בוחרים תחום (שברים, אחוזים, גאומטריה, חשיבה מרחבית ועוד) ופותרים שאלה ב-4 אפשרויות. 3 ניסיונות לכל שאלה. בכל ניסיון שגוי הרמז גדל — קודם משפט אחד מההסבר, אחר כך יותר, ובסוף ההסבר המלא עם התשובה.

**מבחן לדוגמה.** 24 שאלות מתוך אחד משבעת המבחנים, בסדר המקורי שלהן. אופציה לשעון של 60 דקות (כמו במבחן האמיתי). אפשר לדפדף קדימה ואחורה, לסמן שאלות לבדיקה, להחליף תשובות עד הסיום. ציון מלא בסוף, עם פירוט וסקירה של כל שאלה.

**חזרה על טעויות.** כל שאלה שלא הצלחתם בה ב-3 ניסיונות, או ענית עליה לא נכון במבחן, נכנסת לתור חזרה אישי. תוכלו לחזור עליה בכל עת — עד שתפתרו אותה נכון בניסיון הראשון.

**רב-משתמשים.** הוסיפו שם, וההתקדמות שלכם נשמרת רק במכשיר הזה. אחים יכולים להשתמש באותה התקנה — כל אחד עם הסטטיסטיקה שלו. בלי דוא"ל, בלי סיסמה, בלי שרת.

**מותאם לעברית ולכל מסך.** ימין-לשמאל. נוסחאות מתמטיות נכונות. עובד באותה איכות בנייד, בטאבלט ובמחשב.

## למי זה מיועד

לתלמידי כיתה ו׳ שמתכוננים לבחינת הקבלה לתוכנית "נוער מוכשר במתמטיקה" בשכבה ז׳ — או לכל מי שרוצה לתרגל חשבון מתקדם, חשיבה לוגית וחשיבה מרחבית ברמה אתגרית.
